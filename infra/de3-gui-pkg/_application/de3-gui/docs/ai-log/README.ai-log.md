# Goal
- Keep a log of AI interactions for troubleshooting and auditing purposes.

# Instructions
- Logging Activity for code changes under "install-de-gui":
  - When you change code and run commands, create a file here that summarizes the change.
  - The file should be placed in this directory (i.e. <root>.../install-de-gui/ai-log)
  - The file should be in mark-down format
  - The file name should be prefixed with a timestamp formatted as ```date +%Y%m%d%H%M%S```
  - The file name should have a short description of the change,
    less than 80 characters, using only lowercase letters, numbers, and hyphens.
